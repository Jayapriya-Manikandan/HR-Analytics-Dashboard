# 🏢 HR Intelligence Portal

## AI-Powered Employee Attrition Analytics System

A professional HR Analytics web application built using **Python, Streamlit, and Machine Learning**.

This portal helps organizations analyze employee data, understand workforce trends, and predict employee attrition risk using an ML model.

---

## 🚀 Features

### 📊 Interactive Dashboard
- Employee workforce overview
- Department analysis
- Attrition statistics
- Interactive Plotly charts

### 📈 Analytics Module
- Workforce trend analysis
- Department filtering
- Job role analysis
- Income and attrition insights

### 🤖 AI Attrition Prediction
- Machine Learning based prediction
- Employee risk analysis
- Attrition probability score
- Downloadable prediction report

### 📋 Dataset Explorer
- Employee data preview
- Search and filters
- CSV download option

### 📄 HR Reports
- Department reports
- Employee summaries
- Business insights

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Joblib

---

## 🤖 Machine Learning Model

Model Used:

**Random Forest Classifier**

The model predicts whether an employee is likely to leave the organization based on employee-related factors.
